## Installation

I included a script that makes installing them much easier, just follow steps below :

1- Git Clone this repo<br />
2- CD into the theme of choice<br />
3- run `chmod +x ./install.sh`<br />
4- Execute `sudo ./install.sh`<br />

Enjoy ;)
